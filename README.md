# next.js Project Boilerplate

Base on @tannerlinsley's gist [Replacing Create React App with the Next.js CLI](https://gist.github.com/tannerlinsley/65ac1f0175d79d19762cf06650707830) and [colinhacks/nextjs-react-router](https://github.com/colinhacks/nextjs-react-router), with built-in TailwindCSS and custom babel configuration.

## Usage

Using this command create a directory as boilerplate instantly:

```bash
git clone --depth 1 git@github.com:Leechael/nextjs-boilerplate.git [you-project-name] && cd [you-project-name] && rm -rf .git
```
